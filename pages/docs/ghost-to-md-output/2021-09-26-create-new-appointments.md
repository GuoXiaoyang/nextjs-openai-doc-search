![](__GHOST_URL__/content/images/2021/06/new_appointment_-_client.png)

# Edit appointment details

## Select a client

- Search for a client by name
- Add a new client

üí° You can refer to the following information by switching the tabs under the client: - Last/next appointment record - Pet detail - Client contact, payments, and notes
![](__GHOST_URL__/content/images/2021/06/Select_client-1.gif)
## Select pets and services

- Select a **<pet>**
- Select a **<service>**
üí° There will be a Last service mark for your reference
- Select an **<add-on>**
- Adjust the **<Duration>** and **<Price>** if needed
üí° You can save the duration and price for the pet
- Select the **<Staff>** for the service
üí° You can select different staffs for each pet in one ticket
- Add an additional pet for the multi-pet family by clicking the **<+ Additional pet>** button

![](__GHOST_URL__/content/images/2021/06/Select_petservices.gif)
